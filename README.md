# Gerenciando-Instâncias-EC2-na-AWS
Este guia consolida os principais conceitos e práticas para gerenciar instâncias EC2 na AWS, baseado em documentação oficial e boas práticas do mercado.

📌 Visão Geral do EC2
O Amazon Elastic Compute Cloud (EC2) é um serviço web que fornece capacidade de computação redimensionável na nuvem, permitindo que você hospede e gerencie seus sistemas de forma escalável.
